
# Changelog

## [1.2.0] – 2026-01-26

### Added
- Support for **icon-enhanced card titles**
- Flexible icon positioning using CSS Grid:
- Icon above title
- Icon above title (centered)
- Icon inline (left of title)
- Responsive icon layout via `.icon-responsive`
  (icon top on mobile, inline on desktop)
- Dedicated SCSS component for card title layout
  (`.ot-cecard-title` with modifier classes)

### Improved
- Better separation of structure and layout in the card title markup
- Increased layout flexibility without requiring template changes
- Improved visual consistency for cards with and without icons


## [v1.0.0] – 2025-10-10
### Added
- Initial release of the **Card** content element for TYPO3 Sitekit.
- Integration with **ot-irrebuttons** to add configurable button IRRE elements.
- Works standalone or within the **Card Grid** container.
